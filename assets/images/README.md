# Images Folder

## Adding Your Avatar

1. Add your avatar image to this folder as `avatar.png` (or .jpg)
2. The image should be square (recommended: 400x400px or larger)
3. If no image is provided, a default SVG icon will be shown

## Adding Other Images

For blog posts, you can add images here and reference them in your posts:

```markdown
![Image description]({{ '/assets/images/your-image.jpg' | relative_url }})
```

## Supported Formats

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- WebP (.webp)
- SVG (.svg)
